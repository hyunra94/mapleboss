보스 결정석

잡는 보스 선택

보스 이름별로 난이도 하나만 골라요. 선택하면 바로 캐릭터별 집계에 반영돼요.

메이플 알림 감지

공지 전체는 이벤트/점검/업데이트/캐시샵 키워드를 저장하고, 썬데이 탭은 제목에 ‘썬데이’가 들어간 공지만 따로 모아요.

수동 메모

API / 데이터

데이터는 이 기기 브라우저 localStorage에 저장돼요. 깃허브/클라우드플레어로 배포해도 기기별 저장이에요.

웹앱 설치

설치 버튼이 안 뜨면 브라우저 메뉴에서 ‘홈 화면에 추가’ 또는 ‘앱 설치’를 선택해줘. GitHub Pages처럼 HTTPS 주소에서 열어야 설치가 가능해요.

Maplescouter 링크 규칙

캐릭터명만 바꿔서 아래 구조로 열어요.

/ko/result?name=캐릭터명&preset=00000
/ko/info?name=캐릭터명&preset=00000
/ko/spec-order?name=캐릭터명&preset=00000
